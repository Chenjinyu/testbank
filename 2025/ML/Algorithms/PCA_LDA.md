
### 🔹 **Principal Component Analysis (PCA)**

**Type**: Unsupervised dimensionality reduction.

**Goal**:

* Reduce the number of features by projecting data into new axes (**principal components**) that capture the most **variance** in the dataset.

**How it works**:

1. Standardize data.
2. Compute covariance matrix.
3. Extract eigenvectors (directions) and eigenvalues (variance explained).
4. Keep top *k* components to form a smaller feature set.

**Use cases**:

* Preprocessing before ML models.
* Noise reduction.
* Visualization (e.g., compress high-dimensional data into 2D/3D).


### 🔹 **Latent Dirichlet Allocation (LDA)**

**Type**: Unsupervised probabilistic topic modeling.

**Goal**:

* Discover **hidden topics** in a collection of text documents.
* Each document is represented as a mixture of topics, and each topic is a distribution over words.

**How it works** (simplified):

1. Assume documents are generated by choosing topics, then choosing words from those topics.
2. Reverse-engineer this process to infer:

   * Topic-word distributions (which words belong to which topics).
   * Document-topic distributions (which topics appear in which documents).

**Use cases**:

* Topic modeling for large text corpora.
* Document clustering/classification.
* Recommendation systems (e.g., grouping similar articles).

---

### 🔄 **Key Differences Between PCA and LDA (Latent Dirichlet Allocation)**

| Aspect          | **PCA**                                          | **LDA (Latent Dirichlet Allocation)**                |
| --------------- | ------------------------------------------------ | ---------------------------------------------------- |
| **Type**        | Linear algebra (dimensionality reduction)        | Probabilistic modeling (topic modeling)              |
| **Input data**  | Numeric/tabular (features, datasets)             | Text documents (bag-of-words, counts)                |
| **Output**      | New uncorrelated features (principal components) | Hidden topics + topic-word + doc-topic distributions |
| **Objective**   | Capture maximum variance                         | Uncover latent semantic structures                   |
| **Supervision** | Unsupervised                                     | Unsupervised                                         |
| **Domain**      | General ML, statistics                           | NLP, text mining                                     |


✅ **In short**:

* **PCA**: Compresses **numeric features** into fewer dimensions while keeping most variance.
* **LDA (Latent Dirichlet Allocation)**: Finds **hidden topics** in text data, treating documents as mixtures of topics.
